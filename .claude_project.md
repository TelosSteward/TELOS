## Key Technical Concepts

### Primacy Attractor (PA):
- Point in embedding space (ℝⁿ, typically n=1536 or 3072)
- Represents conversational purpose
- Established via calibration dialogue
- Reference point for all subsequent responses

### Fidelity Measurement:
- F = cos(E_response, PA)
- F ∈ [-1, 1]
- F ≥ 0.7 typically "in basin" (acceptable)
- F < 0.7 triggers intervention

### Intervention Mechanism:
1. Halt response delivery
2. Inject PA context into system message
3. Regenerate with reinforced purpose
4. Measure new fidelity F'
5. Deliver if F' ≥ threshold, else escalate

### Lyapunov Stability (Optional Enhancement):
- L(state) = 1 - cos(E_state, PA)
- Stable if L(t+1) ≤ L(t) (energy decreasing)
- Intervention on trend, not just threshold

### Statistical Process Control (SPC):
- Control charts for fidelity tracking
- Process capability analysis (Cpk for governance)
- Out-of-control signals
- Variance reduction as governance metric
